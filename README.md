### Installing
Install Wordpress

After installation, you need to create 2 pages:

- Main page
- Page with airports (URL Slug -> 'properties')

### Wordpress settings
Go to Administration panel -> Settings
- Reading Settings -> Static Page (choose your Main page)
- Permalink settings -> Post name
